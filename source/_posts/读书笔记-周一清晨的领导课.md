世界上任何人都是影响别人和被别人影响的。

影响别人行为的行为，谓之领导。

影响别人行为的能力，谓之领导力。领导力的高低决定一个人生存能力的高低。

领导力不是天生的，而是可藉由训练而来……

## 领导与领导力

记住：你并非孤单一人。
- 谈到领导问题，不仅仅是你会遇到这个困难。大部分人在从雇员转为经理、从经理变为领导者时，都会面临同样的困难。
- 不要垂头丧气，这只会浪费你宝贵的时间。

你所要做的是订立计划，让情况尽量发生转机。任何转变都不会太迟，但你必须努力做出改进。

如果你想获得非凡的成就，必须要做的第一件事便是结束平庸。

周一清晨会谈的基本原则
```
准时开始，准时结束。
实话实说。
尝试不同的方式。
```
## 第一周 司机与乘客

遇到的问题
```
我的下属工作积极性不高。
我想方设法的让下属喜欢我。
我对上级主管工作不满意。
我的业绩使大家沮丧。
```
- 司机与乘客的责任不同。从雇员转为经理或从经理变为领导者，要做出不同的转变。
  - 尽管权利增加了，但同时也失去了过去可以享受的某些自由。
  - 不能再因为自己部门的问题责备他人，而应当承担起全部的责任。
  - 不但要为自己的时间负责，还要为他人的时间负责。
- 真正的领导者只会花时间来解决问题，而不是责备别人。
  - 当你横加指责时，你的注意力集中在过去。如果你承担了责任，你才会将眼光投向未来。
  - 你应当控制自己来应对环境。如果你不在责备甚至避免这个词从你口中说出来，你就会积极的做出某些改变了。

## 第二周 要事第一
“要事”是我们的目标以及需要优先考虑的事情。
三件事情最为首要
```
告诉雇员获取成功的方法和手段。
为顾客提供出色的服务。
盈利。
```
- 什么是要事？
  - 下属们会有形形色色的理解。
- 如果，你想靠别人的领会来达到你的期望，最终你会感到失望。
- 只有团队中的每个人都知道哪些事情最关键，才能将精力置于重要的事务之上，才会是一个专注的、富有效率的工作群体。
- 在多数情况下，老板是员工离职的主要原因。
  - 在炒公司鱿鱼之前，他们已经先炒了上司的鱿鱼。
- 像管理下属一样来管理你的老板。
  - 清楚彼此的期望；
  - 清楚彼此的要事；
  - 清楚双方应当如何彼此协助。

## 第三周 逃离管理黑洞

- ”管理黑洞“
  - 简单的事情变得复杂，人们被蒙蔽了双眼；
  - 人们会因为只说经理爱听的话而受到褒奖；
  - 利己主义盛行，真理不复存在；
  - 对局外人来说，可以将管理黑洞描述为一片混乱、灰心丧气，有时又有点滑稽可笑。
- 学会逃离管理黑洞，与下属保持联系
  - 雇佣优秀的员工；
  - 指导每个员工，让他们变得更优秀；
  - 解雇那些在其位不谋其政的员工。

三类雇员

| 员工分类 | 表现 | 占比 |
| :------- | :------------------------------------ | :--- |
| 超级明星 | 富有经验和学识，想在岗位上发挥最佳作用的人                   | 30%  |
| 中层星   | 也许还不具备超级明星所拥有的经验，或者也许以前是超级明星，由于某些原因，现在失去了争第一的动力 | 50%  |
| 流星     | 尽量避免责任，希望担子越轻越好                               | 20%  |

- 你的职责
  - 并不是通过关照业绩最差的雇员，来降低最低限。
  - 应当通过承认并嘉奖超级明星来提高最上限。

## 第四周 视批评为礼物

批评是贯穿我们一生的学习工具，能教给我们确实的教训

- 了解每个人都要负起责任，包括自己在内
- 重点关注那些建设性而不是攻击性的批评
- 接受建设性批评，将之视为礼物
- 认可批评是一种学习工具，它能终生给我们以教益

## 第五周 做正确的事

- 为长期的、持久的成功，而不是为短期的方便来提高最高标准。
  - 需要建立一套大家都必须遵守的行为准则。
  - 需要有准确的反馈意见。
  - 需要根据雇员所做的决策，积极的或者消极的，实施相应的办法。

这一切都需要你鼓起勇气，坚持正确的做法。

- 做正确的事的原则：即使没人看见也要做正确的事！
- 做正确的事并非轻而易举，事实上有时真的很难。
- 如果现在你没有一套行为准则或者业绩标准的话，就很难知道怎样做才是对的。

你可以采取痛楚最小的方式：忽视问题而不是采取正确的行动。但事实上问题不会自行消失。你必须用正确的方式来解决问题。

- 解决问题的唯一方法：直接查找原因并解决问题。
- 维护正直的形象，就好像他是宝贵的财富一样。
  - 藉由正直才能够建立以及保持别人对你的信任，这是人际关系的根本所在。
  - 你所做的任何事情都会产生影响，因为员工们在看着呢，他们要靠你来树立做正确的事的榜样。
  - 忽视问题是在拿自己正直的口碑冒险。

做正确的事的确需要坚韧不拔的毅力，需要守纪律、重承诺、有勇气。

## 第六周 只雇佣合适的员工

- 你应该在招聘时精挑细选，让加入你的团队成为一种特权。
  - 在面试前精心准备
  - 让其他人参与招聘过程，避免犯感情用事的错误
  - 每个职位都至少要面三个符合条件的应聘者，每个人面试三次，要有三个人对他们进行评估
- 不要为了填补一个职位空缺而降低标准，以后你会为此付出代价的。如果你精挑细选，以后管理起来就会容易的多。

## 第七周 热忱拥抱变化

- 成功应对变化很大程度上取决于对待变化的态度
- 变化让我们退出舒适区，进入进步区
- 人们会抗拒变化，对此我们必须予以理解，并帮助他们更加积极的经历变化

## 第八周 效率！效率！

- 你要想办法统筹自己的时间，这样就可以控制自己的生活了
  - 想办法一点点的挤时间。把零碎时间加在一起，就有更多的时间可以支配了
  - 加快工作节奏

时间是由我们所做的事情以及我们如何做事两者共同决定的。

| 主要的事务，做法正确，举例：召开有效的必要的会议 | 主要的事务，做法错误，举例：再一次重要的会议中浪费了两个小时时间 |
| -------------------- | -------------------- |
| 错事做对，举例：把一次没必要的会议开好 | 错事做错，举例：在一次没必要召开的会议上浪费每个人的时间 |

- 理清事务的轻重缓急并进行合理统筹。
  - 最重要的技巧：每天都留出一段不被干扰的时间做计划
  - 检查放入文件柜中的每份报告
  - 清理办公桌
  - 对回复电子邮件进行控制
  - 集中处理，把事情放在一起做
- 减少干扰
  - 即使你没法阻止别人的干扰，也可以缩短他们干扰你的时间
  - 重新摆放办公设施
  - 与下属和老板安排一对一的会谈
  - 问问下属：“我做的那些事情浪费了你们的时间？妨碍了你们的工作”
- 有效的管理会议
  - 不要染上“永远都定期召开的会议”综合征
  - 要以最重要的事情作为开场白
  - 人们迟到时，不要重复你已经讲过的事情
  - 最简单最有效的方法：准时开会，准时结束
- 花时间读一本时间管理的书籍，找找其他节约时间的方法。

## 第九周 把桶装满的艺术

- 领导者应该牢记不忘的两项内容
  - 作为领导者，你的业绩是团队共同努力的结果
  - 你对团队的需要甚于团队对你的需要
- 每个人都有一桶动力
- 作为一名领导者，你要每个人的桶里都是满满的
- 你是主要的装桶人，装桶的最佳方法是良好的沟通
- 要让下属的动力桶保持充盈必须做到的四件事
  - 需要知道哪些是要事，这对做好工作很重要
  - 向提桶的人反馈意见，告诉他们应该怎么做
  - 让下属知道你很在意他们，也很关心他们的工作
  - 让团队知道他们凝聚为一体有多好

## 第十周 进入遗产地带

- 遗产地带的三个房间：
  - 阅览室：读书越多，知识越丰富。把读书放在首位，你积累的知识，会让你成为升职的首要人物。
  - 倾听室：经理们失败的主要原因在于他们骄傲自大，过于自负，麻木不仁。
  - 给予室：付出越多，回报越大，记得相互给予很重要。
